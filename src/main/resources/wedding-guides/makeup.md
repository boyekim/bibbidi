# 메이크업 준비 점검 가이드

## 사용 목적

사용자가 메이크업 준비 상태를 물으면 저장된 계약과 리허설 일정을 확인하고 누락된 인원, 시간, 출장·얼리스타트 비용을 질문한다.

## 필수 점검 항목

### 서비스 대상

- 완료 기준: 신랑·신부, 혼주, 추가 인원의 헤어와 메이크업 범위를 확인했다.
- 부족 안내: 대상 인원별 시작 시간과 비용을 나누어 확인하도록 안내한다.
- 후속 질문: "신랑·신부 외에 혼주나 추가 인원도 예약하셨나요?"

### 본식 시간과 이동

- 완료 기준: 메이크업 시작 시간, 종료 예상 시간, 웨딩홀 이동 시간을 확인했다.
- 부족 안내: 예식 시작에서 이동·환복 시간을 역산해야 한다고 설명한다.
- 후속 질문: "예식 시작 시간에 맞춘 메이크업 시작 시간을 안내받으셨나요?"

### 리허설과 상담

- 완료 기준: 사전 상담 또는 리허설 날짜, 원하는 스타일과 참고 이미지를 준비했다.
- 부족 안내: 피부·헤어 특이사항을 사전에 공유하도록 제안한다.
- 후속 질문: "메이크업 리허설이나 사전 상담 일정을 잡으셨나요?"

### 추가 비용

- 완료 기준: 얼리스타트, 출장, 원장 지정, 헤어 변형, 혼주 추가 비용을 확인했다.
- 부족 안내: 기본 계약금과 당일 추가 가능 비용을 구분해 기록한다.
- 후속 질문: "얼리스타트나 출장, 원장 지정에 추가 비용이 있나요?"

### 수정과 준비물

- 완료 기준: 당일 수정 메이크업, 헤어 변형 횟수, 준비물과 세안 안내를 확인했다.
- 부족 안내: 당일 담당자 연락처와 지참물을 정리하도록 안내한다.
- 후속 질문: "당일 수정 메이크업과 헤어 변형은 몇 회 포함되나요?"

## 응답 예시

"업체와 총 금액은 등록되어 있지만 리허설 일정, 혼주 포함 범위, 얼리스타트 비용은 미확인이에요. 먼저 리허설이나 사전 상담 일정을 잡으셨나요?"

## 응답 규칙

1. 데이터에 없는 항목은 `미확인`으로 표현하고 준비 여부를 추측하지 않는다.
2. 예식 시간과 이동 시간을 가장 먼저 연결해서 확인한다.
3. 누락 항목은 최대 세 개만 안내한다.
4. 가장 중요한 후속 질문 하나로 응답을 마친다.
5. 리허설이나 상담 날짜는 일정 후보로 구조화한다.
